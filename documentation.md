# Basics

## Windows

### The main editor window
The main editor window consists of 3 main panels as well as the toolbar. The main panel, the game scene is the centered panel in which the game environment takes place in. The left hierachy window is the list of current entities present within the game scene. And finally, the inspector located on the right hand side is the configuration manager that controls and edits the properties of entities.

### The "Asset Manager"
The asset manager is the main library where your sprites, audio, etc lives. You are able to simply add, remove and rename aswell as configure assets within this menu as it is very straightforward.

### The "Global Variables Dictionary"
The GVD is a centralised window where the variables used throught the project live.
